# SistemaRiegoAutomatico

Este proyecto consiste en una interfaz gráfica desarrollada en C# con Windows Forms, diseñada para el control y monitoreo de un sistema de riego automático para plantas. La aplicación permite gestionar distintos actuadores como luces, bomba de agua y una alarma sonora, además de supervisar el nivel de humedad del suelo en tiempo real.

El sistema cuenta con dos modos de funcionamiento:

Modo Manual: El usuario controla cada actuador mediante botones de encendido y apagado.

Modo Automático: El sistema activa los actuadores según los niveles de humedad recibidos desde sensores externos.

La interfaz está optimizada para ser intuitiva y visualmente atractiva, con indicadores gráficos que muestran claramente el estado de cada elemento del invernadero. La comunicación con el microcontrolador se realiza a través de puerto serial, permitiendo una integración directa con hardware real.
